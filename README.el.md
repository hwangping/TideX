# TideX

Παλίρροιες σε πραγματικό χρόνο και πρόβλεψη για κάθε ακτογραμμή στον κόσμο

Ζωντανή εφαρμογή: [https://tide-x.vercel.app](https://tide-x.vercel.app)

## Επισκόπηση

Το TideX είναι πλέον μία ενιαία εφαρμογή παλίρροιας που αναπτύσσεται από τη ρίζα του αποθετηρίου. Για τους χρήστες του GitHub αυτό σημαίνει μία βάση κώδικα για υπολογιστή, κινητό browser και εγκατάσταση στην αρχική οθόνη.

## Κύρια χαρακτηριστικά

- Επιλέξτε οποιοδήποτε παράκτιο σημείο στον χάρτη ή μεταβείτε απευθείας στην τρέχουσα τοποθεσία σας.
- Βρείτε κοντινές ονομασμένες παραλίες από δεδομένα OpenStreetMap.
- Δείτε καμπύλη παλίρροιας 24 ωρών με ώρες πλημμυρίδας και γραμμές αναφοράς 15 ημερών.
- Συγκρίνετε ύψος παλίρροιας, ριπές ανέμου και κατεύθυνση ανέμου στην ίδια προβολή.
- Πλοηγηθείτε σε παρελθοντικές, τρέχουσες και μελλοντικές ημερομηνίες με αρμονικό fallback έξω από το παράθυρο άμεσων δεδομένων.
- Χρησιμοποιήστε την ίδια responsive PWA σε desktop, mobile web και αρχική οθόνη.

## Δωρεάν πηγές δεδομένων

- Open-Meteo Marine: σειρές παλίρροιας / στάθμης θάλασσας
- Open-Meteo Forecast: ταχύτητα ριπών και κατεύθυνση ανέμου
- Πλακίδια OpenStreetMap: βασικός χάρτης
- Overpass API: εύρεση κοντινών παραλιών
- Nominatim: αντίστροφη γεωκωδικοποίηση επιλεγμένων σημείων

## Μέθοδος πρόβλεψης

1. Χρησιμοποιήστε άμεση σειρά παλίρροιας όταν υπάρχει για το επιλεγμένο σημείο και ημερομηνία.
1. Εξομαλύνετε την ημερήσια καμπύλη με μονότονη κυβική παρεμβολή.
1. Συμπληρώστε μερικά κενά με αρμονική συμπλήρωση.
1. Επιστρέψτε σε αρμονικό μοντέλο παλίρροιας όταν λείπουν άμεσα δεδομένα.
1. Βελτιώστε τις ώρες πλημμυρίδας με ανίχνευση τοπικών ακροτάτων και τετραγωνική παρεμβολή.

## Εγκατάσταση ως εφαρμογή

1. Ανοίξτε το [https://tide-x.vercel.app](https://tide-x.vercel.app) σε Safari, Chrome ή άλλο σύγχρονο browser.
1. Χρησιμοποιήστε `Προσθήκη στην αρχική οθόνη` ή `Εγκατάσταση εφαρμογής` από το μενού του browser.
1. Εκκινήστε το TideX σαν κανονική εφαρμογή με το ίδιο web deployment.

## Διεθνοποίηση

- Τα πακέτα γλώσσας runtime βρίσκονται στο `locales/` και φορτώνονται από JSON.
- Το αποθετήριο περιλαμβάνει 42 γλώσσες, μαζί με RTL υποστήριξη για αραβικά, εβραϊκά και ουρντού.
- Μετά την επεξεργασία των γλωσσικών αρχείων εκτελέστε `node scripts/generate-locales.mjs` για αναδόμηση του `locales/index.json`.


## Τοπική ανάπτυξη

Σερβίρετε τη ρίζα του αποθετηρίου με οποιονδήποτε static file server:

```bash
cd TideX
python3 -m http.server 5173
```

Έπειτα ανοίξτε το `http://localhost:5173`.

## Ανάπτυξη

- Αναπτύξτε τη ρίζα του αποθετηρίου σε Vercel, Netlify, Cloudflare Pages ή οποιοδήποτε static host.
- Δεν απαιτείται build βήμα.
- Η root εφαρμογή περιλαμβάνει ήδη PWA manifest, εικονίδια και service worker.

## Αποποίηση ευθύνης

- Το TideX προορίζεται για σχεδιασμό και κατανόηση ακτογραμμής, όχι για πιστοποιημένη ναυσιπλοΐα.
- Οι πραγματικές συνθήκες μπορεί να διαφέρουν λόγω πίεσης, κυματισμού, ροής ποταμών, τοπικής βαθυμετρίας και καιρού.
