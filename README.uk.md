# TideX

Поточні та прогнозні припливи для будь-якого узбережжя світу

Онлайн-застосунок: [https://tide-x.vercel.app](https://tide-x.vercel.app)

## Огляд

TideX тепер є єдиним вебзастосунком припливів, що розгортається з кореня репозиторію. Для користувачів GitHub це означає одну кодову базу для десктопа, мобільного браузера та встановлення на головний екран.

## Ключові можливості

- Обирайте будь-яку прибережну точку на мапі або одразу переходьте до поточного місця.
- Знаходьте названі пляжі поруч за даними OpenStreetMap.
- Переглядайте 24-годинну криву припливу з часами припливу та 15-денними еталонними лініями.
- Порівнюйте висоту припливу, пориви вітру та напрям вітру в одному поданні.
- Працюйте з минулими, поточними й майбутніми датами з гармонічним fallback поза вікном прямих даних.
- Використовуйте ту саму адаптивну PWA на десктопі, мобільному вебі та головному екрані.

## Безкоштовні джерела даних

- Open-Meteo Marine: ряди припливу / рівня моря
- Open-Meteo Forecast: швидкість поривів і напрям вітру
- Тайли OpenStreetMap: базова мапа
- Overpass API: пошук найближчих пляжів
- Nominatim: зворотне геокодування вибраних точок

## Підхід до прогнозу

1. Використовуйте прямий ряд припливу, коли він доступний для вибраної точки та дати.
1. Згладжуйте денну криву монотонною кубічною інтерполяцією.
1. Часткові прогалини заповнюйте гармонічним доповненням.
1. За відсутності прямих даних переходьте до гармонічної моделі припливу.
1. Уточнюйте часи припливу локальним пошуком екстремумів і квадратичною інтерполяцією.

## Встановлення як застосунку

1. Відкрийте [https://tide-x.vercel.app](https://tide-x.vercel.app) у Safari, Chrome або іншому сучасному браузері.
1. Використайте `Додати на головний екран` або `Встановити застосунок` з меню браузера.
1. Запускайте TideX як звичайний застосунок, зберігаючи той самий вебдеплой.

## Інтернаціоналізація

- Мовні пакети runtime лежать у `locales/` і завантажуються з JSON.
- Репозиторій містить 42 мови, включно з RTL-підтримкою для арабської, івриту та урду.
- Після редагування мовних файлів запустіть `node scripts/generate-locales.mjs`, щоб перебудувати `locales/index.json`.

## Структура проєкту

```text
TideX/
├─ index.html
├─ styles.css
├─ app.js
├─ locales/
├─ icons/
├─ manifest.webmanifest
├─ service-worker.js
├─ offline.html
├─ scripts/
│  ├─ generate-locales.mjs
│  └─ generate-readmes.mjs
├─ README.md
└─ README.<locale>.md
```

## Локальна розробка

Віддайте корінь репозиторію будь-яким сервером статичних файлів:

```bash
cd TideX
python3 -m http.server 5173
```

Потім відкрийте `http://localhost:5173`.

## Розгортання

- Розгортайте корінь репозиторію на Vercel, Netlify, Cloudflare Pages або будь-якому статичному хості.
- Крок збірки не потрібен.
- Кореневий застосунок уже містить PWA manifest, іконки та service worker.

## Застереження

- TideX призначений для планування та розуміння прибережних умов, а не для сертифікованої навігації.
- Реальні умови можуть відрізнятися через тиск, хвилювання, річковий стік, локальну батиметрію та погоду.
