# TideX

גאות ושפל בזמן אמת ובתחזית לכל קו חוף בעולם

האפליקציה החיה: [https://tide-x.vercel.app](https://tide-x.vercel.app)

## סקירה

TideX היא כעת אפליקציית גאות ושפל אחת שמופעלת ישירות משורש המאגר. עבור משתמשי GitHub, המשמעות היא בסיס קוד יחיד למחשב שולחני, לדפדפן נייד ולהתקנה על מסך הבית.

## עיקרי היכולות

- בחרו כל נקודת חוף על המפה או עברו מיד למיקום הנוכחי שלכם.
- אתרו חופים בעלי שם בקרבת מקום על בסיס נתוני OpenStreetMap.
- צפו בעקומת גאות ושפל של 24 שעות עם זמני גאות וקווי ייחוס ל-15 יום.
- השוו בין גובה הגאות, משבי הרוח וכיוון הרוח באותה תצוגה.
- עברו בין תאריכים בעבר, בהווה ובעתיד עם גיבוי הרמוני מחוץ לחלון הנתונים הישירים.
- השתמשו באותה PWA רספונסיבית במחשב, בדפדפן נייד ובמסך הבית.

## מקורות נתונים חינמיים

- Open-Meteo Marine: סדרות גאות / מפלס ים
- Open-Meteo Forecast: מהירות משבים וכיוון רוח
- אריחי OpenStreetMap: מפת בסיס
- Overpass API: חיפוש חופים קרובים
- Nominatim: גיאוקידוד הפוך לנקודות שנבחרו

## שיטת החיזוי

1. להשתמש בסדרת גאות ישירה כשהיא זמינה לנקודה ולתאריך שנבחרו.
1. להחליק את עקומת היום באמצעות אינטרפולציה קובית מונוטונית.
1. להשלים פערים חלקיים באמצעות השלמה הרמונית.
1. לעבור למודל גאות הרמוני כאשר אין נתונים ישירים.
1. לדייק את זמני הגאות הגבוהה באמצעות זיהוי קיצונים מקומיים ואינטרפולציה ריבועית.

## התקנה כאפליקציה

1. פתחו את [https://tide-x.vercel.app](https://tide-x.vercel.app) ב-Safari, Chrome או בדפדפן מודרני אחר.
1. השתמשו ב-`Add to Home Screen` או `Install App` מתפריט הדפדפן.
1. הפעילו את TideX כמו אפליקציה רגילה תוך שמירה על אותו דיפלוי וובי.

## בינאום

- חבילות השפה בזמן ריצה נמצאות ב-`locales/` ונטענות מקבצי JSON.
- המאגר כולל 42 שפות, כולל תמיכת RTL בערבית, עברית ואורדו.
- לאחר עריכת קובצי השפה הריצו `node scripts/generate-locales.mjs` כדי לבנות מחדש את `locales/index.json`.

## מבנה הפרויקט

```text
TideX/
├─ index.html
├─ styles.css
├─ app.js
├─ locales/
├─ icons/
├─ manifest.webmanifest
├─ service-worker.js
├─ offline.html
├─ scripts/
│  ├─ generate-locales.mjs
│  └─ generate-readmes.mjs
├─ README.md
└─ README.<locale>.md
```

## פיתוח מקומי

הגישו את שורש המאגר בעזרת כל שרת קבצים סטטי:

```bash
cd TideX
python3 -m http.server 5173
```

לאחר מכן פתחו את `http://localhost:5173`.

## פריסה

- פרסו את שורש המאגר ל-Vercel, Netlify, Cloudflare Pages או כל אחסון סטטי אחר.
- אין צורך בשלב build.
- אפליקציית השורש כבר כוללת manifest, אייקונים ו-service worker של PWA.

## הצהרת אחריות

- TideX מיועדת לתכנון ולהבנת תנאי חוף, ולא לניווט מוסמך.
- התנאים בפועל עשויים להשתנות בגלל לחץ, גלים, זרימת נהרות, בתימטריה מקומית ומזג אוויר.
