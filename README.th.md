# TideX

TideX คือเว็บแอปน้ำขึ้นน้ำลงระดับโลกที่ออกแบบแบบ mobile-first ผู้ใช้สามารถเปิดผ่านเบราว์เซอร์มือถือได้ทันที (ปัจจุบัน deploy บน Vercel) เลือกจุดชายฝั่งใดก็ได้บนแผนที่ แล้วดูแนวโน้มระดับน้ำทะเลพร้อมเวลาน้ำขึ้นสูงที่แม่นยำในแต่ละวัน

## การเข้าถึงออนไลน์

- Production (Vercel): `https://tide-x.vercel.app`

## แนวทางของโปรเจกต์

- กลุ่มผู้ใช้หลัก: โต้คลื่น ตกปลาริมฝั่ง ท่องเที่ยวชายทะเล กิจกรรมท่าเรือ
- คุณค่าหลัก: เลือกจุดได้ทั่วโลก กราฟอ่านง่าย ใช้งานลื่น ไม่ต้องติดตั้ง
- รูปแบบการ deploy: static frontend ล้วน โฮสต์ได้บน Vercel/GitHub Pages/Netlify

## ฟีเจอร์หลัก

- เลือกจุดชายฝั่งทั่วโลกบนแผนที่
- ตรวจจับตำแหน่งผู้ใช้อัตโนมัติ
- แสดงรายชื่อชายหาดใกล้เคียงจาก OpenStreetMap
- กราฟน้ำขึ้นน้ำลง 24 ชั่วโมง
- ทำเครื่องหมายเวลา high tide บนกราฟ
- เลือกวันและเวลาย้อนหลัง/ปัจจุบัน/อนาคต
- ใช้ harmonic prediction อัตโนมัติเมื่อไม่มีข้อมูลตรงของวันที่เลือก
- รองรับหลายภาษา 12 ภาษา

## ประสบการณ์บนมือถือ

- เลย์เอาต์ responsive สำหรับ iPhone และ Android
- ปรับการสัมผัสสำหรับแผนที่ สไลเดอร์ และปุ่ม
- ดีไซน์ทันสมัย อ่านง่ายแม้อยู่กลางแจ้ง
- frontend เบาและตอบสนองเร็ว

## แหล่งข้อมูลและตรรกะการพยากรณ์ (ฟรีทั้งหมด)

- ชุดข้อมูลระดับน้ำทะเล/น้ำขึ้นน้ำลง: Open-Meteo Marine API
- แผนที่พื้นฐาน: OpenStreetMap
- จุดสนใจชายหาด: OpenStreetMap Overpass API
- Reverse geocoding: Nominatim
- การพยากรณ์: ทำ harmonic fitting จากข้อมูลย้อนหลังเมื่อไม่มีข้อมูลตรง

## อัลกอริทึมการพยากรณ์ (สรุป)

TideX ใช้แนวทาง 2 ขั้น: ใช้ข้อมูลตรงก่อน และใช้โมเดลสำรองเมื่อข้อมูลไม่พอ

- ใช้ข้อมูลตรงก่อน: หากมีข้อมูลระดับน้ำทะเลรายชั่วโมงของวันที่เลือก จะใช้ข้อมูลนั้นก่อน
- ทำเส้นโค้งให้ละเอียดขึ้น: เพิ่มความถี่ข้อมูลด้วยการอินเตอร์โพเลชันแบบ cubic monotone
- พยากรณ์สำรอง: หากไม่มีข้อมูลตรง จะฟิตโมเดลฮาร์มอนิกจากข้อมูลย้อนหลัง
- วิธีฟิตโมเดล: ใช้องค์ประกอบน้ำขึ้นน้ำลงหลายชุด (sin/cos basis) ด้วย least squares
- เวลา high tide: ตรวจจับจุดสูงสุดเฉพาะที่และปรับละเอียดด้วย quadratic interpolation

## เทคโนโลยีที่ใช้

- Frontend: HTML + CSS + JavaScript
- แผนที่: Leaflet
- กราฟ: ECharts
- เวลา: Luxon
- Deploy: Vercel (static hosting)

## การพัฒนาในเครื่อง

```bash
cd TideX
python3 -m http.server 5173
```

เปิด: `http://localhost:5173`

## คำแนะนำการตั้งค่า Vercel

- Framework Preset: `Other`
- Build Command: เว้นว่าง (เว็บไซต์ static)
- Output Directory: เว้นว่าง (deploy จาก root)
- ควรเปิด HTTPS เพื่อให้การขอสิทธิ์ตำแหน่งเสถียรขึ้น
