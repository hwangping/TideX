# TideX

ดูน้ำขึ้นน้ำลงแบบเรียลไทม์และคาดการณ์สำหรับชายฝั่งทุกแห่งทั่วโลก

แอปออนไลน์: [https://tide-x.vercel.app](https://tide-x.vercel.app)

## ภาพรวม

TideX เป็นเว็บแอปน้ำขึ้นน้ำลงแบบชุดเดียวที่เผยแพร่จากโฟลเดอร์รากของรีโพแล้ว สำหรับผู้ใช้ GitHub นี่หมายถึงโค้ดชุดเดียวสำหรับเดสก์ท็อป เบราว์เซอร์มือถือ และการติดตั้งลงหน้าจอหลัก

## จุดเด่น

- เลือกจุดชายฝั่งใดก็ได้บนแผนที่หรือกระโดดไปยังตำแหน่งปัจจุบันได้ทันที
- ค้นหาชายหาดที่มีชื่อใกล้เคียงจากข้อมูล OpenStreetMap
- ดูกราฟน้ำขึ้นน้ำลง 24 ชั่วโมงพร้อมเวลาน้ำขึ้นสูงและเส้นอ้างอิง 15 วัน
- เปรียบเทียบระดับน้ำ ความแรงลมกระโชก และทิศทางลมในมุมมองเดียว
- ดูวันที่ย้อนหลัง ปัจจุบัน และอนาคต พร้อม fallback แบบฮาร์มอนิกเมื่ออยู่นอกหน้าต่างข้อมูลตรง
- ใช้ PWA แบบ responsive เดียวกันได้ทั้งเดสก์ท็อป เว็บมือถือ และหน้าจอหลัก

## แหล่งข้อมูลฟรี

- Open-Meteo Marine: ชุดข้อมูลน้ำขึ้นน้ำลง / ระดับน้ำทะเล
- Open-Meteo Forecast: ความเร็วลมกระโชกและทิศทางลม
- OpenStreetMap tiles: แผนที่พื้นฐาน
- Overpass API: ค้นหาชายหาดใกล้เคียง
- Nominatim: reverse geocoding สำหรับจุดที่เลือก

## แนวทางการคาดการณ์

1. ใช้ชุดข้อมูลน้ำขึ้นน้ำลงตรงก่อน เมื่อมีข้อมูลสำหรับจุดและวันที่เลือก
1. ทำให้กราฟรายวันเรียบขึ้นด้วย monotone cubic interpolation
1. เติมช่วงข้อมูลที่ขาดบางส่วนด้วย harmonic completion
1. หากไม่มีข้อมูลตรง จะถอยกลับไปใช้โมเดลน้ำขึ้นน้ำลงแบบฮาร์มอนิก
1. ปรับเวลา高潮ด้วยการตรวจจับค่าสูงสุดเฉพาะที่และการแทรกค่าแบบกำลังสอง

## ติดตั้งเป็นแอป

1. เปิด [https://tide-x.vercel.app](https://tide-x.vercel.app) ใน Safari, Chrome หรือเบราว์เซอร์สมัยใหม่อื่น
1. ใช้ `เพิ่มไปยังหน้าจอหลัก` หรือ `ติดตั้งแอป` จากเมนูเบราว์เซอร์
1. เปิด TideX ได้เหมือนแอปทั่วไปโดยยังคงใช้ deployment เว็บเดียวกัน

## การทำให้รองรับหลายภาษา

- ชุดภาษาระหว่างรันอยู่ใน `locales/` และโหลดจาก JSON
- รีโพนี้มี 42 ภาษา รวมถึงการรองรับ RTL สำหรับอาหรับ ฮีบรู และอูรดู
- หลังแก้ไขไฟล์ภาษา ให้รัน `node scripts/generate-locales.mjs` เพื่อสร้าง `locales/index.json` ใหม่

## โครงสร้างโปรเจกต์

```text
TideX/
├─ index.html
├─ styles.css
├─ app.js
├─ locales/
├─ icons/
├─ manifest.webmanifest
├─ service-worker.js
├─ offline.html
├─ scripts/
│  ├─ generate-locales.mjs
│  └─ generate-readmes.mjs
├─ README.md
└─ README.<locale>.md
```

## พัฒนาในเครื่อง

เสิร์ฟโฟลเดอร์รากของรีโพด้วย static file server ใดก็ได้:

```bash
cd TideX
python3 -m http.server 5173
```

จากนั้นเปิด `http://localhost:5173`

## การเผยแพร่

- ดีพลอยโฟลเดอร์รากไปยัง Vercel, Netlify, Cloudflare Pages หรือ static host ใดก็ได้
- ไม่ต้องมีขั้นตอน build
- แอปรากได้รวม PWA manifest, ไอคอน และ service worker ไว้แล้ว

## ข้อจำกัดความรับผิดชอบ

- TideX ใช้สำหรับการวางแผนและทำความเข้าใจสภาพชายฝั่ง ไม่ใช่เครื่องมือนำทางที่ได้รับการรับรอง
- สภาพจริงอาจต่างออกไปจากความกดอากาศ คลื่นลม การไหลของแม่น้ำ ภูมิประเทศใต้น้ำเฉพาะพื้นที่ และสภาพอากาศ
