# TideX

Текущие и прогнозные приливы для любого побережья мира

Онлайн-приложение: [https://tide-x.vercel.app](https://tide-x.vercel.app)

## Обзор

TideX теперь разворачивается только из корня репозитория. Для пользователей GitHub это одна кодовая база для десктопа, мобильного браузера и установки на главный экран.

## Возможности

- Выбирайте любую прибрежную точку на карте или сразу переходите к своему местоположению.
- Находите ближайшие именованные пляжи по данным OpenStreetMap.
- Просматривайте 24-часовую кривую прилива с отметками времени прилива и 15-дневными линиями сравнения.
- Сравнивайте высоту прилива, порывы ветра и направление ветра в одном представлении.
- Работайте с прошлыми, текущими и будущими датами, а вне окна прямых данных используется гармонический fallback.
- Используйте одну и ту же адаптивную PWA на десктопе, в мобильном браузере и на домашнем экране.

## Бесплатные источники данных

- Open-Meteo Marine: ряды прилива / уровня моря
- Open-Meteo Forecast: скорость порывов и направление ветра
- Тайлы OpenStreetMap: базовая карта
- Overpass API: поиск ближайших пляжей
- Nominatim: обратное геокодирование выбранных точек

## Подход к прогнозу

1. При наличии прямого ряда прилива для выбранной точки и даты используется он.
1. Суточная кривая сглаживается монотонной кубической интерполяцией.
1. Частичные пробелы заполняются гармоническим дополнением.
1. Если прямые данные недоступны, используется гармоническая модель прилива.
1. Время прилива уточняется через поиск локальных экстремумов и квадратичную интерполяцию.

## Установка как приложения

1. Откройте [https://tide-x.vercel.app](https://tide-x.vercel.app) в Safari, Chrome или другом современном браузере.
1. Выберите `Добавить на главный экран` или `Установить приложение` в меню браузера.
1. Запускайте TideX как обычное приложение, сохраняя тот же веб-деплой.

## Интернационализация

- Языковые пакеты времени выполнения лежат в `locales/` и загружаются из JSON.
- Репозиторий включает 42 языка, включая RTL-поддержку для арабского, иврита и урду.
- После редактирования языковых файлов запустите `node scripts/generate-locales.mjs`, чтобы пересобрать `locales/index.json`.

## Локальная разработка

Поднимите корень репозитория любым сервером статических файлов:

```bash
cd TideX
python3 -m http.server 5173
```

Затем откройте `http://localhost:5173`.

## Развёртывание

- Разворачивайте корень репозитория на Vercel, Netlify, Cloudflare Pages или любом статическом хостинге.
- Шаг сборки не требуется.
- Корневое приложение уже включает PWA-манифест, иконки и service worker.

## Отказ от ответственности

- TideX предназначен для планирования и понимания береговой обстановки, а не для сертифицированной навигации.
- Реальные условия могут отличаться из-за давления, зыби, речного стока, локальной батиметрии и погоды.
